## What you get

- **Major AI Releases** — New models and announcements as they drop
- **Important Developments** — Industry moves worth knowing about
- **New Tools & Products** — Frameworks, APIs, and dev tools you can use today
- **Open Source Momentum** — Trending projects before everyone else finds them
- **Early Signals** — What's emerging before it hits the mainstream
- **Important Research** — Only the papers that matter outside academia

Every item is scored, ranked, and explained in plain English — so you know *why* it matters, not just *that* it happened.

## How it works

Aggregates 15+ sources every 30 minutes — official lab blogs (OpenAI, Google AI, DeepMind), top tech press (TechCrunch, The Verge, Ars Technica), research (arXiv, HuggingFace), and open source (GitHub Trending).

Items are automatically scored on importance, novelty, and real-world relevance. Research papers are deprioritized unless they have practical impact.

## Self-hosting

```bash
npm install
npm run db:seed    # Initialize database
npm run dev        # http://localhost:3000
```

See `.env.example` for configuration options.
